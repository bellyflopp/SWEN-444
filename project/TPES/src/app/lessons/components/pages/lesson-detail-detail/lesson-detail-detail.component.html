 Working with dynamic data

To create a variable in Javascript, you will need to "declare" the variable.

var x = 0;

There are also other variables that you can declare other than numbers. To declare a list of objects, use an array object.

var myList = [];

You can also statically declare strings in javascript. You do it like this.

var myString = "Hello, world!";

And remember, ALL Javscript code statements end in a semicolon!
