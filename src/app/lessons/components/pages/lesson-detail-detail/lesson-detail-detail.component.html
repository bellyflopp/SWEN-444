To create a variable in Javascript, you will need to "declare" the variable.

var x = 0;

There are also other variables that you can declare other than numbers. To declare a list of objects, use an array object.

var myList = [];

You can also statically declare strings in javascript. You do it like this.

var myString = "Hello, world!";

And remembers, ALL Javascript code statements end in a semicolon!

CSS, also known as Cascading Style Sheets, is a way for you to style your web pages in a clean manner

CSS has what are known as 'classes', which tell CSS what styles to apply and where.

{{".myStyle { color: red;}"}}

There are a lot of other attributes you can declare too, but we will get into those in the next lesson.

For now, just know that you need to include braces around all of the classes you right

HTML, also known as Hypertext Markup Language, is the structure of your website

It determines what items are placed on the page, and in what order

HTML tags contain classes, which is how you apply your css to your structure.

For example, if you hit F12 on your keyboard right now, you will be able to see the HTML of this page!
